# ADR-002: 認証方式 — GitHub OAuth実名制

**状態:** 確定
**日付:** 2026-06-30

## 決定

GitHub OAuthで全員実名ログイン。匿名にしない。

## 根拠

- G's Academyの受講生は全員GitHubアカウントを持っている
- 匿名掲示板ではなく、チューター/メンターと相談するプロダクト。相手が誰か分かる必要がある
- Better Auth + GitHub OAuthはbs-job-boardのGS-9で設計済み。知見を流用

## ロール

| ロール | 説明 | 設定方法 |
|---|---|---|
| student | G's受講生 | デフォルト |
| tutor | チューター | 管理者が手動設定 |
| mentor | メンター | 管理者が手動設定 |

## 集計

シンプルな集計を入れる。利用頻度、相談カテゴリ別傾向、チューター/メンター応答件数。
