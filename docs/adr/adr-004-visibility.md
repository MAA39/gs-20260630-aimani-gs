# ADR-004: 相談の公開範囲 — 本人選択制

**状態:** 確定
**日付:** 2026-06-30

## 決定

相談の公開範囲は本人が選ぶ。

## 公開範囲の選択肢

| 範囲 | 説明 |
|---|---|
| 自分だけ | AIと整理するだけ。誰にも見えない |
| チューターに見せる | 指定したチューターに相談文を共有 |
| メンターに見せる | 指定したメンターに相談文を共有 |
| 全体に出す | 受講生全員に公開（共感・コメント可） |

## 不可侵ルール

- アプリを使った人間を絶対に不利にしない
- 本人が選ばない限り、外に出さない
- AIが勝手に共有しない
- 共有前に本人が確認・編集できる
